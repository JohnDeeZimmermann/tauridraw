# Tauridraw

This project is a desktop version of Excalidraw using Tauri. 
It is highly customized and specifically tuned to my individual preferences and needs.

The src/excalidraw-app directory contains the Excalidraw React source code. 
The src-tauri/ directory contains the React Tauri backend. 

We use Yarn as our build system. 

## Code Guidelines
We strive for simple code that is easy to reason about, maintainable and bug free. 
Don't make too many assumptions and use proper typing wherever possible.
